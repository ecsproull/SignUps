$qty )

When money is to be collected we transfer control back to Stripe.com and they do the credit card processing.

As that process proceeds, Stripe sends events back to us to let us know about the progress.

Parameters
string$descriptionDescription of what is bing paid for..
int$price_idThe price ID to be charged to.
int$badgeAttendee's badge number.
int$attendee_idThe id fo the entry in the attendees table.
int$costThe dollar cost of the signup.
int$qtyThe quantity being purchased.

◆ payment_event()

payment_event ( )

Called by Stripe.com when a payment has been processed.

There are several events but we only register for a few of them. Event registration is done on the Stripe.com web site when you set up the account.

Only verify the event if there is an endpoint secret defined Otherwise use the basic decoded event
