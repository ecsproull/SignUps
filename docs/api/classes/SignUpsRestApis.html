Creates a form that displays the rolling sessions along with their attendees.

protected create_rolling_session_select_form2(string $signup_name, array<string|int, mixed> $attendees, int $signup_id, object $template, array<string|int, mixed> $template_items, string $user_group, bool $admin, string $secret, string $description, mixed $template_items2[, mixed $rolling_days = null ]) : void

This is a long function and studying the code is the only way to fully understand it. It allows members to select slots and also remove themselves from a previously selected slot. Some of the signups, such as monitors are not allowed to remove themselves within 2 days of their duty time. By default the user can see the default number of days that the signup specifies. Usually 30 days. There is a drop down where 60 and 90 days can be selected. The dropdown also allows to see the past 30 days to look at the history.

There is a lot of logic intertwined with the HTML that is being generated which makes this a hard function to follow. Good luck.
